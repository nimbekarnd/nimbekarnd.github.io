UV Sterilization Robot Path Planning

Developed a Voronoi-based Coverage Path Planning (VP-CPP) approach using Spanning Tree Coverage (STC) for UV sterilization robots. The system achieved 98% dose-qualified coverage compared to 88% baseline, reduced mission time by 27%, and decreased path overlap by 34% across 25 diverse test room configurations. This research addresses the critical need for effective autonomous UV disinfection in healthcare environments.

Coverage: 98%
Time Reduction: 27%
Path Efficiency: 34% less overlap
Python Path Planning Voronoi Diagrams Coverage Algorithms STC

Autonomous Wall Following Robot

Implemented an autonomous wall-following behavior for TurtleBot3 using ROS and Python. Tested extensively in Gazebo simulation environments, demonstrating reliable performance in various corridor configurations and obstacle scenarios.

Dijkstra's Algorithm Visualization

Interactive visualization tool for understanding Dijkstra's pathfinding algorithm using OpenCV and Python. The system provides real-time visual feedback as the algorithm explores the search space, highlighting the optimal path and visited nodes. Features customizable map configurations, obstacle placement, and adjustable start/goal positions for educational and research purposes.

Implemented genetic algorithms, constraint optimization techniques, and various heuristics to find optimal routes that balance distance minimization with charging station visits.
